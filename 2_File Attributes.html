File-Attributes :-

1. Name :-

- only information kept in human-readable form :- A String of alpha-numeric character and special symbol like (_) used by user to refer file.

2. Identifier- Unique Tag of file :-

- Uniquely identifies the file within the file system. Used by OS to refer file.

3. Type - needed for systems that support different types :-

- it generally expressed in the form of a file extension. For example, prog.cpp to indicate c++ file.

4. Location - pointer to file location on device :-

- Pointer to the device and location on the device of the file.

5. Size-current file size :-

- Specifies the size of file in bytes, word or blocks. And also maximum allowed size of file.

6. Protection :-

- It determines who can read file, write file, execute file and so on.

7. Usage count :-

- Indicates number of processes that are currently using this file.
