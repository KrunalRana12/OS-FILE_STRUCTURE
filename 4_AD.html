File-Attributes of Advantages :-

File attributes offer several advantages in terms of organizing, managing, and securing data within a computer system. Here are some key advantages:

1.Metadata Information

2.File Organization

3.Access Control

4.Data Integrity

5.File Versioning

File-Attributes of Disadvantages :-

1.Limited Standardization

2.Security Concerns

3.Potential for Misuse

4.Maintenance Overhead

5.Compatibility Issues

File-Operation of advantages :-

1.Easy To Create and Find Operation

2.Allow to Reaname of already set name of file

3.Dletion of created file and realloc storage

4.write and read Easy in file

5.Easy access file under file Manager

File-Operation of Disadvantages :-

1.Time Consuming When We Find Aperticular File

2.File fragmentation

3.Limiting File Versioning

4.Backup and recovery Challenges

5.Concurrency issue
